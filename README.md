# NYC Collision Analysis Dashboard – Power BI

## Project Overview

This project analyzes New York City traffic collision data using Power BI. The goal is to explore collision patterns, identify high-risk areas, and present insights through an interactive dashboard.

## Tools Used

- Power BI
- Excel
- Data Cleaning
- Data Visualization
- Data Analysis

## Dataset

The dataset includes information about traffic collisions in New York City, such as dates, locations, contributing factors, and injury-related details.

## What I Did

- Reviewed and explored the collision dataset.
- Cleaned and organized the data for analysis.
- Built an interactive Power BI dashboard.
- Created visualizations to show collision trends and patterns.
- Analyzed contributing factors and high-risk locations.
- Presented insights in a clear and business-friendly format.

## Business Value

This analysis can help organizations understand traffic collision trends, identify high-risk locations, improve safety planning, and support data-driven decision-making.

## Key Skills Demonstrated

- Power BI dashboard development
- Data cleaning
- Data analysis
- Data visualization
- Reporting
- Business intelligence thinking

## Project Summary

This project helped me practice transforming raw data into meaningful insights. It also improved my ability to create dashboards that communicate information clearly to support better decisions.
